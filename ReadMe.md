## ⚖️ Uygulama Anayasası (The Core Rules)
1. **Miras Yapı:** Tüm uygulamalarda `sinav.db` (SQLite) dosyası temel mimari referansımızdır.
2. **Karakter Standardı:** Kendi projelerinizde ve genişletme çalışmalarınızda **TÜRKÇE KARAKTER KULLANIMI YASAKTIR.** - Örn: `öğrenci_adı` ❌ -> `student_name` ✅
3. **Teknoloji Yığını:** Tasarım için **StarUML**, uygulama için **Google Colab**, portfolyo için **GitHub**.
4. **Veri Kaynağı:** Veritabanlarımızı **Hugging Face** üzerinden çekeceğimiz gerçek dünya veri setleri ile canlandıracağız.

2. İçerideki `sinav.db` dosyasını SQLite Browser veya paylaşılan Colab şablonu ile açın.
